# TechPro Solutions – PC Store Website

A modern and fully responsive PC store website built using HTML, CSS, and JavaScript.
This project features a clean dark-themed UI designed for computer shops, gaming PC businesses, and tech service companies.

* Clean grid layouts

* Font Awesome
